== Description ==

Vireo Property Management is a comprehensive solution for managing rental properties directly from your WordPress dashboard. Whether you manage a single property or hundreds, our plugin provides the tools you need to streamline your operations.

= Free Features =

* **Property Management** - Add unlimited properties with detailed information
* **Unit Management** - Manage individual units within properties
* **Tenant Database** - Keep track of all tenant information
* **Lease Tracking** - Monitor active and expired leases
* **Maintenance Requests** - Handle maintenance tickets efficiently
* **Basic Reporting** - Generate essential property reports
* **Import/Export** - Migrate data from other systems
* **REST API** - Build custom integrations

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/vireo-property-management` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to the Property Management menu to start adding your properties
4. Configure the plugin settings under Property Management > Settings

== Frequently Asked Questions ==

= Can I import my existing property data? =

Yes! We provide CSV import functionality for properties, units, and tenants. We also offer migration guides for popular property management platforms.

= Is this plugin suitable for commercial properties? =

Yes, the core plugin works great for residential and commercial properties. We also offer a Commercial Property add-on with specialized features.

= Can tenants submit maintenance requests online? =

Yes, using our shortcodes or blocks, you can create forms for tenants to submit maintenance requests directly from your website.

= Does this integrate with accounting software? =

The Pro version includes QuickBooks integration, and our Advanced Accounting add-on provides even more options.

= Is there a limit on properties or tenants? =

No! The free version supports unlimited properties and tenants.
