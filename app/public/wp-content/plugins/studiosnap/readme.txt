== Frequently Asked Questions ==

= Is StudioSnap suitable for solo photographers? =
Absolutely! StudioSnap is perfect for solo photographers and scales up to larger studios with multiple photographers.

= Can clients book sessions online? =
Yes! StudioSnap includes a complete online booking system that integrates seamlessly with your WordPress website.

= Does it work with any WordPress theme? =
Yes, StudioSnap is designed to work with any properly coded WordPress theme. The interface adapts to your site's styling.

= Can I customize the booking form? =
Yes, the booking forms are fully customizable and can be styled to match your website design.

= Is there a client portal? =
Yes! Clients get access to a secure portal where they can view their sessions, galleries, and manage their bookings.

= Can I manage multiple types of photography sessions? =
Yes, you can create unlimited session types (portraits, weddings, events, etc.) with different durations and pricing.

= Does it handle payment processing? =
Basic invoice generation is included in the free version. Full payment processing is available in StudioSnap Pro.

= Can I send automated emails to clients? =
Yes! The plugin includes email automation for booking confirmations, reminders, and gallery notifications.

= Is there mobile support? =
Yes, the entire system is mobile-responsive, including the admin interface and client-facing components.

= How do I upgrade to Pro? =
Pro licenses are available at vireodesigns.com with additional features like payment processing, advanced scheduling, and premium support.
