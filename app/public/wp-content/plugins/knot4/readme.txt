== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/knot4` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to the Knot4 menu to start setting up your nonprofit platform
4. Configure your organization details under Knot4 > Settings
5. Set up your first donation form using the [knot4_donation_form] shortcode

== Frequently Asked Questions ==

= Is Knot4 really free? =

Yes! The core plugin is completely free and includes essential donation management, basic CRM, and event features. Pro features are available with our affordable annual plans.

= Do you take a percentage of donations? =

No! Unlike many donation platforms, Knot4 charges a flat annual fee. You keep 100% of your donations (minus standard payment processing fees from Stripe/PayPal).

= Can I import my existing donor data? =

Yes! Knot4 includes CSV import functionality to help you migrate from other systems. Pro version includes direct imports from popular platforms.

= Does it work with my existing theme? =

Absolutely! Knot4 is designed to work with any properly coded WordPress theme. Forms and portals inherit your theme's styling.

= Is it secure for handling donations? =

Yes! Knot4 follows WordPress security best practices and integrates with PCI-compliant payment processors like Stripe and PayPal.

= Can donors manage their own information? =

With the Pro version, yes! The member portal allows donors to update their profiles, view giving history, and manage recurring donations.
