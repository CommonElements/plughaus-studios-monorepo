Professional sports league management system for WordPress. Manage teams, players, schedules, standings, and statistics with ease.

== Description ==

Vireo Sports League Manager is a comprehensive solution for managing sports leagues directly from your WordPress dashboard. Whether you run a youth soccer league, adult basketball tournament, or multi-sport recreational program, our plugin provides the tools you need to streamline your operations.

= Free Features =

* **Team Management** - Create and manage unlimited teams with rosters
* **Player Profiles** - Track player information and statistics
* **Schedule Management** - Create season schedules with venues and times
* **Standings Tables** - Automatic standings calculation based on results
* **Match Results** - Record scores and track game outcomes
* **Basic Statistics** - Player and team performance tracking
* **Public Display** - Shortcodes for schedules, standings, and stats
* **Multi-Sport Support** - Configure for any sport type

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/vireo-sports-league` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to the Sports League menu to start setting up your league
4. Configure your sport type and league settings under Sports League > Settings
5. Add teams, players, and create your schedule

== Frequently Asked Questions ==

= Can I manage multiple leagues or seasons? =

Yes! The plugin supports multiple leagues and seasons. You can archive past seasons and maintain historical data.

= Does it work with any sport? =

Yes, the plugin is designed to be flexible and can be configured for any team sport including soccer, basketball, baseball, hockey, volleyball, and more.

= Can teams register and pay online? =

The Pro version includes full online registration with payment processing through WooCommerce, Stripe, or PayPal.

= Can I customize the appearance? =

Yes! The plugin includes customizable templates and works with any WordPress theme. The Pro version includes additional styling options.

= Is there a limit on teams or players? =

No! The free version supports unlimited teams, players, and seasons.

= Can players or coaches update their own information? =

The Pro version includes frontend dashboards for teams and players to manage their own profiles and view schedules.
